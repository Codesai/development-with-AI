# Prompting

A comprehensive series to experiment on how the structure of a prompt matter.

## Objective

Learn how to progressively improve your prompts by incorporating key techniques: adding role definitions, providing context, setting constraints, structuring formats, including examples, and ultimately building reusable prompt templates.

## Exercises

1. [**01-Naive-Prompting**](01-Naive-Prompting/01-Naive-Prompting.md)  
   Start with basic prompts and understand their limitations.

2. [**02-Role**](02-Role/02-prompting-with-role.md)  
   Add role-based instructions to guide AI behavior and perspective.

3. [**03-Context**](03-Context/03-prompting-with-context.md)  
   Provide relevant context to improve understanding and accuracy.

4. [**04-Constraints**](04-Constraints/04-prompting-with-constraints.md)  
   Set clear constraints to limit scope and guide output.

5. [**05-Format**](05-Format/05-prompting-with-format.md)  
   Specify output format to get structured, consistent results.

6. [**06-Examples**](06-Examples/06-prompting-with-examples.md)  
   Include examples to demonstrate desired behavior and outcomes.

7. [**07-Reusable-Prompts**](07-Reusable-Prompts/07-reusable-prompts.md)  
   Build prompt templates for consistent reuse across projects.
