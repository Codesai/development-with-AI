# 06 - Examples

## Goal

See how providing an example acts as a catch-all, filling in gaps that role, context, constraints, and format alone cannot fully capture.

## Part 1 — Provided example

1. Take the prompt from exercise 05 (role + context + task + constraints + format).
2. Add the user story in [example-user-story.md](example-user-story.md) as an example.
3. Run it and note what changed.

## Part 2 — Your own example

1. Replace the provided example with a user story from your own project.
2. Run it and compare the output with Part 1.
3. Discuss the output with your pair/group.

## Reflection questions

- What did the AI pick up from the example that you never explicitly specified?
- Did the output include anything you didn't ask for but that was present in the example?
- How did the output differ between Part 1 and Part 2?
- What does this tell you about the influence of examples relative to the other prompt elements?
