# 05 - Format

## Goal

See how specifying a format removes ambiguity about the structure of the output and makes it immediately usable.

## Instructions

1. Take the prompt from exercise 04 (role + context + task + constraints).
2. Add format instructions to control the structure of the output.
3. Run it and compare the output with what you got before.
4. Share your prompt and output with the group.

## The Task

Same as before: write a user story for a login feature — but this time tell the AI exactly how the output should be structured.

Think about:
- Should it use the "As a / I want / So that" structure?
- Should it include acceptance criteria? In what format?
- What output format do you want?

## Reflection questions

- Did the output structure change compared to exercise 04?
- Did the AI follow your format instructions exactly?
- Was there anything in the output you still didn't expect?
- How does a consistent format help when working with a team?
