# Step 1 - Manual Test Review

This step contains an implementation and test suite generated by an AI agent. We have intentionally left the AI's output exactly as it was generated to simulate a typical scenario where the AI produces tests that are green but flawed.

## Goal

Review the generated tests as a human reviewer. The tests pass successfully, but your job is to determine whether they truly prove the requested behavior and its boundaries.

## Prompt given to AI

> Implement shipping cost calculation where premium users and non-premium users with packages under 5 kg get free shipping, and packages of 20 kg or more are classified as heavy shipping.

## Run

```bash
cd 01-Test-Review-And-Mutation/Step1
dotnet test
```

## Task

- Read [`src/Shipping/ShippingCostCalculator.cs`](src/Shipping/ShippingCostCalculator.cs).
- Read [`tests/Shipping.Tests/ShippingCostCalculatorTest.cs`](tests/Shipping.Tests/ShippingCostCalculatorTest.cs).
- Perform a **manual review** of the tests:
  - Check whether premium users and non-premium users under 5 kg get free shipping.
  - Check whether heavy shipping is tested.
- Note down which tests are missing and whether the green tests give enough confidence to accept the AI-generated change.
