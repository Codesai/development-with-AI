# Evaluating Results

Exercises for practicing evaluation of AI-generated code, tests, and architectural changes.

## Exercises

- [Exercise 1 - Review an AI-Generated Change](01-review-ai-generated-change.md)
- [Exercise 2 - Mutation Testing Thinking Exercise](02-mutation-testing-thinking.md)
- [Exercise 3 - Architectural Fitness Mini-Lab](03-architectural-fitness-mini-lab.md)

All exercise states are available as folders in `main`. Students do not need to switch branches.

## Exercise Folders

- `01-Testing/Step1`: starting point without meaningful tests.
- `01-Testing/Step2`: AI-generated implementation and tests to review.
- `01-Testing/Step3`: completed review solution for comparison.
- `02-Mutation/Step1`: AI-generated code, tests, and Stryker.NET setup.
- `02-Mutation/Step2`: expected mutation-testing answers and learning notes.
- `03-Architecture/Step1`: generated code with an architectural violation.
- `03-Architecture/Step2`: same code with a failing architecture fitness test.
- `03-Architecture/Step3`: corrected design with the architecture fitness test passing.
