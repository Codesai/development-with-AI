# Exercise 2 - Mutation Testing Thinking Exercise

## Goal

Understand why green tests can be weak.

## Duration

25 minutes

## Setup

Use the prepared folders in `main`. Students do not need to copy or create the code manually.

## Steps

- `02-Mutation/Step1`: predict surviving mutations, run the tests, and run Stryker.NET. Read `02-Mutation/Step1/README.md`.
- `02-Mutation/Step2`: compare your predictions with the expected answers and learning notes. Read `02-Mutation/Step2/README.md`.

The goal is to see why a green AI-generated test suite can still miss important faults.
