# 03 - Code is context

## Goal

Understand that the project code is potentially part of the context for the IA

## Instructions

Do the email validations like in the previous exercises following the same design rules.
Remove all the project instructions from the project (agents.md files)
Remove all skills 
Ask the IA agent to perform validation in the Name, for example a maximun number of characters
 
## Recomendations 

Check if the IA agent follow the same desing rules for the Name validations that are present in the code for the email validation

Your code is context for the agent, if your project follow some clear and consistent rules for validations the agent is probably going to follow them
