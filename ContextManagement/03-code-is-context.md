# 03 - Code is context

## Goal

Understand that the project code is potentially part of the context for the AI.

## Instructions


Do the email validations as in the previous exercises, following the same design rules.
Remove all the project instructions from the project (AGENTS.md files).
Remove all skills.
Ask the AI agent to perform Name validation, for example a maximun number of characters.
 
## Recommendations

Check if the AI agent follows the same design rules for Name validations as the existing in the code for the email validation.

Your code provides context for the agent, if your project follows clear and consistent rules for validations, the agent is probably going to follow them.
